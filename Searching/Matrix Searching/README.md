## Matrix Search

This folder contains problems related to searching patterns
in two-dimensional matrices.

These problems often involve navigating rows and columns efficiently
while handling boundary conditions and neighbor comparisons.

---

## Common Techniques
- Brute force traversal
- Binary search on rows or columns
- Directional guarantees in grids
- Divide and conquer strategies

---

## Typical Problems
- Find a peak element in a 2D grid
- Search in row-wise and column-wise sorted matrices
- Matrix-based optimization problems
