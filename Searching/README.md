## Searching

This folder contains problems and implementations related to searching
techniques commonly used in Data Structures and Algorithms.

Searching problems focus on efficiently locating elements or determining
optimal values under given constraints.

---

## Topics Covered
- Linear Search
- Binary Search
- Binary Search on Answer
- Searching in sorted arrays and matrices
- Optimization-based searching problems

---

## Key Concepts
- Divide and conquer strategies
- Monotonic conditions enabling binary search
- Feasibility checks for optimization problems
- Time and space complexity trade-offs

---

## Usage
Each subfolder groups problems based on the searching pattern used.
Code implementations are written with clarity and interview readiness
in mind.
